Change Font

1. Copy the font to sdcard

2. Press "Font Settings" button and select "Original handwriting font". If it's already selected, go to 5.

3. Make handwriting font. It takes a few minutes.

4. Set handwriting font if it finishes making it.

5. Press "Select CustomFont" and select the font(on sdcard).

6. Restart your phone via power menu.

Delete Font

1. Press "Delete CustomFont"

2. Select the font and press delete
